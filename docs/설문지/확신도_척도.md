## 실험 내 구현 사양

### 표시 시점
- 각 웨이포인트 도달 시 자동 팝업
- 참가자가 웨이포인트 반경 3m 이내 진입 시 트리거

### UI 구현
- **Beam Pro Only / Hybrid Beam Pro 화면**: 화면 중앙에 질문 + 1-7 버튼 표시
- **Glass Only**: 글래스 화면에 1-7 숫자 표시, 고개 기울임(head tilt) 또는 Beam Pro 터치패드로 선택
- **Hybrid 글래스**: Beam Pro 화면에 표시 (글래스에는 "Beam Pro를 확인하세요" 알림)

### 기록 형식
- 이벤트 타입: `CONFIDENCE_RATED`
- `waypoint_id`: 해당 웨이포인트 ID
- `confidence_rating`: 선택된 값 (1-7)
- `timestamp`: 응답 시각

### 응답 시간 제한
- 제한 없음 (참가자가 원하는 만큼 생각 후 응답)
- 단, 10초 이상 미응답 시 부드러운 리마인더 표시

## 분석 시 유의사항

- **트리거 전후 비교**: WP03, WP05가 불확실성 트리거 지점. 이 지점 직전(WP02, WP04)과 직후(WP04, WP06)의 확신도 변화를 핵심 지표로 사용
- **천장 효과 주의**: 초반 웨이포인트에서 대부분 6-7 응답 시, 변화량 탐지가 어려울 수 있음
